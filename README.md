# EasyGoldGlitch3.0

Plantilla de prompt en español para dirigir un "Consejo de Élite" orientado a diseño de experiencias web premium (especialmente en contextos financieros / trading).

## Contenido

- `prompts/elite_multidisciplinary_digital_product_council.md`: Prompt maestro listo para copiar y usar.
- `prompts/brief_template.md`: Brief estructurado para completar antes de pedir propuesta.

## Uso rápido

1. Completa `prompts/brief_template.md` con tu proyecto.
2. Copia el prompt maestro y reemplaza placeholders (`{{...}}`) con tu brief.
3. Pide salida con enfoque estratégico, visual, conversión y soporte.

## Objetivo

Elevar la calidad de propuestas web para que combinen:

- Percepción premium
- Autoridad financiera
- Claridad comercial
- Reducción de fricción
- Conversión con narrativa y diseño de alto nivel
